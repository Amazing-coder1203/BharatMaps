# India GeoJSON Maps (States & Outer Boundaries)

A clean, high-quality, and up-to-date collection of GeoJSON files for the political boundaries of India. This dataset is ready-to-use for web development, GIS projects, and interactive visualization tools (such as Leaflet, Mapbox, MapLibre, D3.js, and React Simple Maps).

---

## 🗺️ Included Files

### 1. State-level political boundaries (`india-states-v2.geojson`)
*   **Format:** GeoJSON `FeatureCollection`
*   **Features:** 36 States & Union Territories (including the separate boundaries of **Jammu & Kashmir** and **Ladakh**).
*   **Key Property:** State names are mapped under `properties.NAME_1` (e.g., `"Andhra Pradesh"`, `"Maharashtra"`, `"Ladakh"`, etc.).
*   **Resolution & Optimization:** Patched with official state boundary guidelines, cleaned of tiny sliver-polygons/interior holes to balance detail and performance (~11.3 MB).

### 2. Country Outer Boundary (`india-outer-v8.geojson`)
*   **Format:** GeoJSON `FeatureCollection`
*   **Features:** Single multi-polygon representing the outer boundary/outline of India.
*   **Use Case:** Ideal for clipping maps, rendering a clean country outline, or showing a highlight boundary in world maps (~309 KB).

## 🛠️ Usage

You can load these files directly into your project.

### 1. React / ES Modules (using local imports)
If you place these files in your source directory, you can import them directly:

```javascript
import React from 'react';
import { ComposableMap, Geographies, Geography } from 'react-simple-maps';
import indiaStatesGeoJSON from './india-states-v2.geojson';

export default function IndiaMap() {
  return (
    <ComposableMap projection="geoMercator" projectionConfig={{ scale: 1000, center: [78.96, 22.59] }}>
      <Geographies geography={indiaStatesGeoJSON}>
        {({ geographies }) =>
          geographies.map((geo) => (
            <Geography 
              key={geo.rsmKey} 
              geography={geo} 
              fill="#ECEFF1" 
              stroke="#CFD8DC" 
            />
          ))
        }
      </Geographies>
    </ComposableMap>
  );
}
```

### 2. Direct HTTP Fetch / CDN
You can fetch the raw GeoJSON files dynamically in your app using the raw GitHub CDN links:

```javascript
// State boundaries GeoJSON
const STATES_GEOJSON_URL = 'https://raw.githubusercontent.com/Amazing-coder1203/BharatMaps/main/india-states-v2.geojson';

// Outer country boundary GeoJSON
const OUTER_GEOJSON_URL = 'https://raw.githubusercontent.com/Amazing-coder1203/BharatMaps/main/india-outer-v8.geojson';

fetch(STATES_GEOJSON_URL)
  .then(response => response.json())
  .then(geoJsonData => {
    // Load into your Mapbox, Leaflet, or D3 instance
    map.addSource('india-states', {
      type: 'geojson',
      data: geoJsonData
    });
  });
```

## ⚙️ How These Files Were Generated (Patch details)
The J&K and Ladakh boundaries in standard international datasets (like Natural Earth) often do not align with official Indian representations. 

These files were generated by:
1. Extracting the baseline map of India's states.
2. Downloading high-resolution district-level boundaries for J&K and Ladakh from public covid19india mapping files.
3. Merging the districts of each territory using TopoJSON/GeoJSON tools to form two clean, unified state outlines.
4. Patching the baseline map with these updated, merged geometries to produce a seamless, correct national representation.
